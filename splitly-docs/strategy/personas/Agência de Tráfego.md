# Persona 3: A Agência de Tráfego

> ← [[Estratégia de Aquisição]] | Veja também: [[Afiliado Profissional]], [[Produtor Lançador]]

---

## Perfil Demográfico

| Campo | Detalhe |
|-------|---------|
| Idade | 28–45 anos |
| Gênero | Equilibrado |
| Localização | Brasil — capitais |
| Renda mensal | R$ 15.000 – R$ 60.000 (dono) |
| Educação | Ensino superior completo (Marketing, Publicidade, Administração) |

## Perfil Profissional

| Campo | Detalhe |
|-------|---------|
| Ocupação | Dono ou sócio de agência de tráfego pago |
| Experiência | 3–8 anos no mercado |
| Tamanho da agência | 3–15 funcionários |
| Clientes | 5–30 produtores, afiliados ou e-commerces |
| Ticket médio por cliente | R$ 1.500 – R$ 5.000/mês (só gestão de tráfego) |
| Ferramentas atuais | Meta Ads, Google Ads, GA4, Looker Studio, Hotjar/Clarity, CRM próprio |

## Dia a Dia

- Reuniões de resultado com clientes (2-4 por dia)
- Supervisão da equipe de gestores de tráfego
- Análise de relatórios e dashboards
- Prospecção de novos clientes
- Resolução de crises: "O ROAS caiu. O que aconteceu?"
- Passa 40% do tempo em reunião, 30% analisando números, 30% em operação

## Dores

### Dor Principal
**"Toda reunião de resultado o cliente pergunta: 'Mas como você sabe que essa página é a melhor?' Eu mostro print do Meta Ads. Mostro planilha. Nunca é suficiente. O cliente quer PROVA. Quer ver os números com os próprios olhos."**

### Dores Secundárias
- "Tenho 15 clientes. Cada um testa páginas diferentes. É impossível gerenciar isso com planilha."
- "Meus gestores perdem tempo criando campanhas separadas pra cada variação em vez de otimizar o que já funciona."
- "Quando o cliente não entende os resultados, ele cancela. Já perdi 3 clientes esse ano por falta de transparência."
- "Preciso de relatórios profissionais. Looker Studio resolve, mas leva horas configurar."

## Objeções de Compra

| Objeção | Refutação |
|---------|-----------|
| "Já pago um monte de ferramenta." | ROI: 1 cliente retido = R$ 18k-60k/ano. O Splitly se paga com 1 cliente que não cancelar. |
| "Minha equipe vai ter que aprender." | 2 minutos pra configurar. A equipe não muda nada — só usa 1 link em vez de 3. |
| "Preciso de white label." | Domínio próprio incluso. API pública. Relatórios exportáveis. |

## Gatilhos de Decisão

1. **Link público de compartilhamento** — o cliente vê os resultados sem login
2. **API pública** — integra no Looker Studio com 1 clique
3. **Domínio próprio** — `links.agencia.com/r/cliente` — profissionalismo máximo
4. **Relatórios CSV** — exporta e manda pro cliente
5. **Dark mode** — passa credibilidade de produto premium
