# Estratégia de Aquisição — Splitly

> ← [[Splitly - Home]] | Personas: [[Afiliado Profissional]], [[Produtor Lançador]], [[Agência de Tráfego]]

---

## Ordem de Ataque

| Fase | Persona | Quando | Por quê |
|------|---------|--------|---------|
| **Fase 1** | [[Afiliado Profissional]] | Imediato | Maior volume, dor mais aguda, decisão rápida |
| **Fase 2** | [[Produtor Lançador]] | Mês 2-3 | Ticket maior, precisa de case/prova social |
| **Fase 3** | [[Agência de Tráfego]] | Mês 4-6 | MRR previsível, exige API + white label |

---

## Fase 1: Afiliado Profissional (Mês 1-3)

### Estratégia de Canal

```
YouTube (60%) → Review/tutorial → Landing Page → Cadastro → Onboarding → Ativação
Instagram Ads (20%) → Criativo dor/solução → Landing Page → Cadastro
Grupos WhatsApp (10%) → Indicação boca a boca → Cadastro direto
Meta Ads (10%) → Público segmentado → Landing Page → Cadastro
```

### Ações Táticas

| Ação | Detalhe |
|------|---------|
| **Vídeo review no YouTube** | "Testei 3 ferramentas de split de tráfego — só 1 funciona com Hotmart." Mostrar Splitly na prática. |
| **Parceria com criadores** | Carlos Novaes, Amanda Agostinho, Tiago Tessmann. Oferecer acesso gratuito em troca de review. |
| **Grupos de WhatsApp** | Criar grupo "Splitly — Teste A/B de Páginas" e distribuir dicas + acesso antecipado. |
| **Instagram Ads** | Criativo: "Você está testando páginas do jeito ERRADO." → Mostrar como fazer certo com Splitly. |
| **Meta Ads** | Segmentar: interesses em Hotmart, Kiwify, tráfego pago, Meta Ads. |

### Métricas da Fase

| Métrica | Alvo Mês 1 | Alvo Mês 3 |
|---------|------------|------------|
| Usuários cadastrados | 100 | 500 |
| Experimentos ativos | 40 | 250 |
| Cliques processados | 5k | 50k |
| Conversões rastreadas | 50 | 500 |
| Usuários pagos (futuro) | 0 (grátis) | 10-20 |

## Fase 2: Produtor / Lançador (Mês 2-4)

### Estratégia

```
Cases de afiliados → Prova social → Site/landing → Demo → Fechamento
Indicação de afiliados → "Meu afiliado usa Splitly" → Cadastro
Eventos → Networking → Demo → Fechamento
```

### Ações Táticas

| Ação | Detalhe |
|------|---------|
| **Case study documentado** | Pegar 3 afiliados da Fase 1 que tiveram sucesso. Documentar: "Aumentou ROAS em X% usando Splitly." |
| **Página de cases** | Criar `/cases` com resultados reais. Print do dashboard, depoimento em vídeo. |
| **Demo personalizada** | Para produtores com potencial: 15 min de call mostrando o Splitly integrado ao fluxo deles. |
| **Parceria com plataformas** | Conversar com Hotmart, Kiwify sobre integração oficial. |

---

## Fase 3: Agência de Tráfego (Mês 4-6)

### Estratégia

```
LinkedIn → Conteúdo autoridade → Demo → Fechamento
Indicação de clientes → Cliente vê link público → "Que ferramenta é essa?" → Cadastro
Casos de agências → Prova social → Inbound
```

### Ações Táticas

| Ação | Detalhe |
|------|---------|
| **Conteúdo no LinkedIn** | Artigos sobre "Como provar ROI pro seu cliente de tráfego pago" |
| **Whitepaper** | "O Guia Definitivo de Teste A/B para Agências de Tráfego" em troca de email |
| **Plano Agência** | Preço especial para múltiplos clientes. Domínio próprio incluso. API ilimitada. |
| **Integração Looker Studio** | Template pronto de relatório que puxa dados da API do Splitly. |

---

## Plano de Lançamento (30 dias)

```
Semana 1: Lançamento interno
  → Convidar 20 afiliados do grupo pessoal
  → Coletar feedback, bugs, sugestões
  → Gerar primeiros cases

Semana 2: Conteúdo
  → Publicar vídeo review no YouTube
  → Posts no Instagram e LinkedIn
  → Distribuir nos grupos de WhatsApp

Semana 3: Tráfego pago
  → Ativar Meta Ads e Instagram Ads
  → Testar 3 criativos (dor, curiosidade, autoridade)
  → Otimizar pelo criativo campeão

Semana 4: Iteração
  → Analisar métricas da landing page
  → Melhorar copy com base em objeções reais
  → Preparar Fase 2 (cases de sucesso)
```
